# FunnerHot-SantiagoAlcaraz
Funnelhot  Technical test  Name: Santiago Alcaraz

Administrador de Asistentes IA con opcion de Entrenamiento.

es un projecto web de dos paginas  la raiz como homo Administrador de  IA donde hay un CRUD Local  de los asistentes   con un adicion de un boton para entrenar
Segunda pagina "Entrenar" donde se enruta a la raiz+ID del Asistente IA  que trae la informacion de del asistente y donde puedes  1- darle parametros a la IA  y 2- hablar con ella en una simulacion de ChatBot

la Navegacion consiste en el Logo para ir a la Raiz-Home donde se encuentra el administrador con el CRUD  y el Boton de "Entrenar"
